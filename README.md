# 🌌 Transit Method Analysis – Astrophysics Project

## 📘 Problem Overview

This project is based on a **Transit Method analysis problem** in astrophysics, where a distant space probe observes the Sun and planetary transits.

The objective is to compute and analyze various observational parameters such as **flux, angular size, and brightness variation** caused by planetary transits.

### 🔍 Problem Breakdown:

* Estimation of solar flux at a large distance and its bolometric magnitude
* Determination of the angular size of the Sun
* Estimation of the angular size of Jupiter
* Calculation of how much of the Sun is covered during transit
* Change in observed magnitude due to Jupiter’s transit
* Similar magnitude change estimation for Earth’s transit

Each part is solved independently in well-structured Jupyter Notebooks.
